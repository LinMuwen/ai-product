一 项目概述🎯

1.1 项目背景

华图教育是职业教育领域领跑企业，全国超1000家学习中心、万余名员工，正从传统职业考试培训向全周期学习与职业服务延伸。华图已推出AI面试点评、AI申论批改等约20款AI产品，研发投入同比暴涨160%至1.45亿元，覆盖近70%员工的AI工作台。命题的核心课题是：如何面向海量在校学生，实现从"随性摸索"到"路径可落地"的跨越，为不同个体匹配专属成长方案。

当前大学生职业规划面临三重结构性问题：

学情画像不精准：传统人工摸排效率低、维度单一。主流测评工具仅测性格兴趣，从未将家庭经济压力、父母职业资源、生理耐受度等客观条件纳入模型。78.3%的贫困在校生偏好体制内就业，但实际进入比例仅45.4%——脱离家庭约束的规划建议几乎没有落地价值（周宇香，2020）。

定制规划效率低：大学分年级成长节点多、各赛道要求差异大，依靠人工批量产出个性化方案耗时耗力。郑雅君（2023）《金榜题名之后》揭示了"小镇做题家"的"直觉依赖模式"困境——家庭背景弱势的学生在外部指令撤除后，缺乏自主决策的内部系统。

赛道路径拆解难：升学、公考、求职等方向分支繁杂，规则持续更新。更关键的是——没有任何产品利用真实案例数据做相似人群的路径回溯，帮学生优先排除不适配选项。

产品战略意义：将用户触点从"决定考公后才找到华图"前移至"大学入学阶段即开始陪伴"，构建"免费职业探索 → 成长规划 → 日常伴学 → 识别需求 → 课程推荐 → 就业跟踪"的完整用户生命周期。

1.2 产品定位

本产品是一款AI驱动的大学生全周期成长规划引擎，以AI大模型为底层智能核心，以多维学情画像为基础，以动态路径推演与阶段任务引擎为核心能力，覆盖大学4-5年全周期。产品帮助大学生回答三个终极问题："我是谁？我能去哪？下一步怎么走？"核心理念是：不追求给出"最热门"的职业方向，而是帮助学生深入认识到自己可能都没有意识到的真正需求，找到不一定热门但最适合每个人的专属职业目标。用AI专家Agent代替高价人工咨询，做到"小镇做题家也有春天"。

2.2 竞品分析

对12款主流产品进行了系统拆解（求职精灵、职比特、北森、中公AI就业、粉笔、超级简历、牛客网、实习僧、互联派、前途问鹿、山小芽、学职平台），核心结论：

  • 没有任何产品实现"通用规划+多赛道推演+全周期任务跟进"一体化
  • 多数产品脱离现实约束，未将家庭经济、父母职业资源、身体机能等客观约束纳入模型
  • 所有产品聚焦大四求职，忽略大一至大三的启蒙和试错
  • 考培型产品（中公/粉笔）以规划为导流手段，本质卖课——规划是手段而非目的
  • 没有任何产品利用真实案例做相似人群路径回溯
  • 没有任何产品将运动手表生理监测纳入职业规划辅助

三 学术理论支撑📚

3.1 自我决定理论（SDT）

Ryan & Deci提出的SDT认为，自主性需求是行为持续性的前提。Cahyasari（2024）证实自我决定水平与生涯决策能力呈显著正相关（r=0.685）。Lu等（2024）基于1246名中国大学生追踪设计发现：学生需要先有能力想象"自己想成为谁"，单纯的"老师让你干什么"无法替代这种认知建构。

→ 产品启示：规划工具的Step 0必须是"目标澄清"——通过"目的先行"设计强制学生在起点明确"我为什么做规划"，打破直觉依赖模式。

3.2 生涯建构理论（CCT）

Savickas提出的CCT以生涯适应力为核心概念。Gao等（2025）证实"能看见自己的未来路径"本身就是一种保护性心理资源。

→ 产品启示：多赛道可视化推演帮助学生建构未来视角；前人案例参照降低信息不对称。路径推演的目的不是给出唯一最优解，而是刺激学生对自身未来工作自我的认知建构和主动探索。

3.3 家庭因素与文化再生产

Li等（2012）以高考成绩为控制变量，发现"干部父母"的毕业生起薪溢价高达15%。郑雅君（2023）区分了"目标掌控"与"直觉依赖"两种阶层习性。王昳玥等（2021）证实家庭SES通过心理资本完全中介职业探索。Qi等（2024）发现父母期望"直接指令无效"——只有通过激发子女主动探索才产生效果。

→ 产品启示：需多维建模家庭约束，对于来自高期望-强干预家庭的学生，为其保留自主决策的独立空间。

3.4 运动手表（HRV）的学术可行性

Kim等（2024）确认HRV在测量教育应激水平时具有中等共时效度。Benitez-Agudelo等（2025）发现学生"更好的成绩可能以长期身心健康为代价"。

→ 产品启示：用户授权同步的运动手表HRV监测已具备学术可行性，但目前尚无产品将其纳入职业规划辅助——这构成学术创新空间。

四 核心功能设计⚙️
1

AI 成长管家 · 多 Agent 协作

Multi Agent

系统采用"1 个核心成长管家 Agent + N 个专业 Agent"的协作架构。核心 Agent 是学生唯一需要交互的入口，负责理解需求、管理上下文、调度专业 Agent、汇总结果并解释下一步。学生感受到的是一个持续理解自己的成长管家，背后则由多个专家 Agent 共同支持。

核心成长管家 画像与数据分析 职业路径规划 任务与资源编排 进度复盘与校准 关键节点与风险提醒
2

规划执行与动态跟踪

实时监测

四子模块构成完整跟踪闭环：

  • 游戏化进度条：按任务难度系数加权显示
  • 四节点检测：日/周/月/学期 完成率 + 目标偏离 + 匹配漂移 + 预警分级
  • 身心状态报备：运动手表 HRV → 应激检测 → 岗位风险标记
  • 阶段复盘报告：每阶段结束自动生成总结 + 优化建议
3

职途初鉴 · 八维画像

画像系统

八维画像：性格适配度、专业能力、兴趣匹配度、学习与知识、抗压与情绪、沟通协作、健康情况、家庭情况，统一 0-5 分制。

新增 "抗压与情绪"、"沟通协作"、"健康情况"、"家庭情况" 四项业界空白评估维度 —— 规划先问"你能不能"，再问"你想不想"。

4

我的榜样 · 前人案例参照

行业首创

基于八维画像做相似度匹配，推荐"人生榜样"及完整轨迹时间轴。对"直觉依赖型"学生尤其重要。冷启动由华图教研专家标注100-200条典型路径案例。

5

多赛道路径推演引擎

推演核心

覆盖公考、考研、出国、央国企、互联网、金融、教育、创业等8+主赛道。四象限（难度-价值）可视化展示，支持3-4条路径并行对比。冷启动教研专家区间匹配→数据积累后自动切换相似度计算。

6

分年级阶段任务引擎

任务引擎

大一 探索→大二 定向→大三 冲刺→大四 决胜

每项任务附带华图教研资料和推荐课程，学习记录实时回流画像引擎，形成"规划→学习→数据沉淀→画像刷新"闭环。

五 技术架构设计🔧

5.1 整体架构

四层架构：展现层（学生端H5/小程序 + 高校管理后台 + 运营看板）→ 应用层（学情画像/路径推演/任务引擎/AI顾问/数据报告/榜样匹配）→ AI引擎层（LLM对话引擎/画像建模引擎/路径推荐算法/任务生成引擎/多Agent协同）→ 数据层（用户画像库/赛道知识图谱/任务模板库/华图教研库/行为日志/正反案例库）。

5.2 三大AI核心引擎

画像建模引擎：融合游戏化测评 + Agent对话挖掘 + 学习行为数据 + 情感交互记录，构建八维动态画像。通过持续行为日志和用户授权同步的运动手表数据进行增量更新——强调"主动知识沉淀"而非一次性测评。

赛道匹配与路径推演引擎：两阶段算法——冷启动：教研专家标注100-200条典型路径案例+赛道理想画像区间匹配；数据积累后自动切换相似度匹配。技术路线：知识图谱（Neo4j）+ 相似度匹配 + 专家区间标注。

阶段任务生成引擎：赛道 × 年级 × 能力差距 × 节点日历 → LLM生成学期任务清单+推荐资源，支持自然语言动态切换和重新推演。技术路线：LLM + 规则引擎 + 模板库。

八 核心创新点💡
01
多 Agent 协同
9 Agent

9 个 Agent 按职能分层协作，共享底层画像与事件总线，按需调度：

数据采集 画像建模 案例匹配 职业规划 能力提升 性格挖掘 进度监督 情感陪伴 异常预警
02
游戏化冷启动
模拟人生

首次使用时通过"模拟人生"式场景选择小游戏，让学生在沉浸式互动中自然暴露性格偏好、价值取向和能力倾向——告别枯燥问卷。学生在玩游戏的过程中完成画像采集，游戏行为数据作为多Agent的初始输入。

03
数据闭环与持续进化
自进化

从被动询问到主动知识沉淀——学生每次学习、每次提问、每次情绪表达都回流至画像模型。形成"规划→执行→数据沉淀→画像刷新→更精准规划"的自进化闭环。就业后回访数据验证，持续优化推荐准确率。

04
目的先行
范式重构

所有竞品默认学生以"自我实现"为单一目标，本产品强制用户在起点选择"我做职业规划是为了什么"（自我实现/经济独立/摆脱原生家庭/体验多元人生）。SDT证明自主性需求满足直接预测决策质量——这不是增量优化，而是对"职业规划"本身的重新定义：从"帮你找最好的路"→"先帮你想清楚你为什么要走路"。

05
客观约束与主观偏好解耦
行业首创

先按家庭条件、身体机能、精力阈值划定可行性区间，再结合兴趣与价值观调优。杜绝"测出来你适合做咨询但你家掏不起MBA学费"的脱离现实规划。

实现路径：家庭约束旁注运动手表授权数据接入

06
前人案例正向匹配
首创

首创"人生榜样"正向匹配机制：基于八维画像进行相似度匹配，找到与用户背景高度相似且已达成目标的真实案例。展示其完整轨迹时间轴——从大学起点到各关键节点。对"直觉依赖型"学生尤其重要。冷启动：华图教研专家标注100-200条典型案例。

九 预期价值与成果🌟

9.1 商业价值

  • C 端用户会员费：面向个人大学生提供会员订阅服务（99-299 元/年），解锁深度规划报告、动态任务引擎、无限制 AI 顾问等进阶功能

9.2 社会价值

  • 教育公平：AI规划服务打破地域与资源壁垒，缩小"小镇做题家"与优势家庭学生的信息鸿沟

  • 人才培养：缩短学生从校园到职场的适应周期
